- GSAP

## Local Run Guide

1. Clone the repository:
   ```bash
   git clone https://github.com/najim2004/bbstudio-task.git
   ```
2. Navigate to the project directory:
   ```bash
   cd bbstudio-task
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
